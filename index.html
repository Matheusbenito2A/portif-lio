HTML (HyperText Markup Language) Definição: HTML é a linguagem de marcação padrão usada para criar páginas da web. Ela fornece a estrutura básica de uma página, que é então aprimorada e modificada por outras tecnologias como CSS e JavaScript. Função: Define o conteúdo e a estrutura de uma página web. Usa elementos ou tags para descrever diferentes tipos de conteúdo, como parágrafos, links, cabeçalhos, imagens, listas, etc. CSS (Cascading Style Sheets) Definição: CSS é uma linguagem de folha de estilo usada para descrever a apresentação de um documento escrito em HTML. CSS controla o layout de múltiplas páginas da web de uma só vez. Função: Define estilos para elementos HTML, incluindo cores, fontes, margens, alturas, larguras, imagens de fundo, posicionamento avançado e muitos outros. Permite a separação entre o conteúdo (HTML) e a apresentação (design visual). Definição: JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web. É uma linguagem de script orientada a objetos e baseada em protótipos que é executada no lado do cliente (navegador). Função: Adiciona interatividade e comportamento dinâmico às páginas web. Pode ser usado para manipular o conteúdo do HTML, alterar estilos CSS, responder a eventos do usuário, e muito mais. Permite a criação de aplicações web ricas e interativas.
